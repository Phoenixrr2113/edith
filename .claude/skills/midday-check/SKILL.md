# Midday Check

Scan for changes since morning: new emails, afternoon calendar, reminders.

If a meeting is < 4h away, prep now. If a deadline is approaching, advance the work — draft, research, write. If actionable emails arrived, draft replies.

Only message Randy if something needs his attention or approval. If nothing is new or actionable, write to taskboard and stay silent. Do NOT message "nothing to report."
